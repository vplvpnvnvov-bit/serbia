Синхронизация данных (Облако)

Связывает ваши устройства между собой. Данные автоматически сохраняются в облаке при каждом изменении чек-листа.

Ваш код синхронизации: Загрузка...

Статус облака: Нет данных о синхронизации

«Синхронизировать сейчас» принудительно забирает свежие данные из облака или отправляет локальные изменения. «Изменить код» позволяет подключить другое устройство.

🚀

Версия программы (Код приложения)

Само приложение работает автономно. Если разработчики добавили новые функции, карту или исправили ошибки, нужно обновить саму программу.

Текущая версия на устройстве: Загрузка версии...

Скачивает новую версию приложения с сервера (карты, логику, дизайн). Программа перезагрузится сама, если найдет обновление.

⚠️

Опасная зона

Полное удаление всей вашей информации без возможности восстановления.

Внимание! Это действие физически сотрет ваши данные из облака Firebase, очистит память этого телефона и сгенерирует вам новый чистый код.
