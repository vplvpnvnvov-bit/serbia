⚙️ Управление данными и обновлениями

Здесь вы можете управлять сохранностью ваших данных, синхронизировать устройства и обновлять само приложение.

🔄

Синхронизация данных (Облако)

Связывает ваши устройства между собой. Данные сохраняются в облаке только когда вы сами нажмёте «Выгрузить». Чтобы получить данные с другого устройства, нажмите «Загрузить».

Ваш код синхронизации: Загрузка...

Последняя синхронизация: Нет данных

Статус облака: Проверка...

«Выгрузить» — сохраняет ваши данные в облако. «Загрузить» — читает данные из облака на это устройство. «Сгенерировать новый код» — создаёт новый код, старый остаётся в облаке нетронутым. «Изменить код» — подключиться к данным другого устройства. «Удалить данные из облака» — физически удаляет документ из Firebase, код остаётся.

🚀

Версия программы (Код приложения)

Само приложение работает автономно. Если разработчики добавили новые функции, карту или исправили ошибки, нужно обновить саму программу.

Текущая версия на устройстве: Загрузка версии...

Скачивает новую версию приложения с сервера (карты, логику, дизайн). Программа перезагрузится сама, если найдет обновление.

⚠️

Сброс устройства

Очищает все локальные данные (чек-лист, калькулятор, настройки), но сохраняет код синхронизации. Облачные данные не затрагиваются.

После сброса вы можете загрузить свежие данные из облака (кнопка «Загрузить») или начать с чистого листа и выгрузить пустой чек-лист в облако.
